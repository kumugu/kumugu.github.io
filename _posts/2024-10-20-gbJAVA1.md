# JAVA 변수와 연산자



## 1. 자바 클래스와 문법 개요

자바(Java)는 클래스 단위로 프로그램을 작성한다. 프로그램은 클래스 내부에 작성되며, 각 클래스는 첫 글자를 대문자로 시작하는 것이 관례다. 또한 모든 명령어는 세미콜론(`;`)으로 끝나야 한다. 자바의 시작점은 `main()` 메서드이다. 자바 가상 머신(JVM)이 이 메서드를 호출하여 프로그램을 실행하게 된다.

## 1.1 자바 프로그램 실행 과정

1. **소스 파일 작성**: `.java` 확장자로 소스 파일을 작성한다.
2. **컴파일**: 작성된 자바 소스 파일을 컴파일하면 바이트 코드(`.class`)가 생성된다.
3. **바이트 코드 실행**: JVM은 바이트 코드를 실행하여 프로그램을 동작시킨다.
4. **메모리 할당**: 실행 중 JVM이 운영체제에서 메모리를 할당받고, `main()` 메서드를 실행한다.

### 1.2 표준 출력

`System.out`은 자바의 표준 출력 장치를 의미한다. `println()`, `print()`, `printf()` 메서드를 사용하여 콘솔에 출력을 할 수 있다.

- **`println()`**: 출력을 하고 줄바꿈을 한다.
- **`print()`**: 출력을 하고 줄바꿈을 하지 않는다.
- **`printf()`**: 서식 지정 출력을 한다.

```
System.out.println("안녕하세요.");  // "안녕하세요." 출력 후 줄바꿈
System.out.print("Hello");  // "Hello" 출력 후 줄바꿈 없음
System.out.printf("%d + %d = %d\n", 10, 20, 30);  // 서식 지정 출력
```

### 1.3 자바의 제어문자

- **`\t`**: 일정 간격(tab)만큼 띄어쓰기
- **`\n`**: 줄바꿈
- **`\s`**: 공백을 나타내는 제어문자로, 띄어쓰기의 역할을 한다.

```
System.out.println("안녕하세요\t여러분"); // "안녕하세요"와 "여러분" 사이에 탭 간격
System.out.println("Hello\nWorld");  // "Hello"와 "World" 사이에 줄바꿈
System.out.println("Java\sProgram"); // "Java"와 "Program" 사이에 공백
```

## 2. 자바의 변수와 자료형

### 2.1 변수의 개념

변수는 데이터를 저장할 수 있는 메모리 공간이다. 자바에서는 변수를 선언할 때 자료형을 명시해야 하며, 자료형에 따라 저장할 수 있는 값의 종류와 크기가 결정된다.

### 2.2 변수 선언 방법

```
자료형 변수명;
```

- **자료형**: 자바에는 8가지 기본 자료형이 있다.
- **변수명**: 데이터를 저장할 이름으로, 사용자가 임의로 정할 수 있다.

### 2.3 기본 자료형

1. **정수형**: `byte`(1바이트), `short`(2바이트), `int`(4바이트 - 기본형), `long`(8바이트)
2. **실수형**: `float`(4바이트), `double`(8바이트 - 기본형)
3. **문자형**: `char`(2바이트)
4. **논리형**: `boolean`(1바이트)

### 2.4 변수 사용 예시

```
int number = 10;  // 정수형 변수 선언 및 초기화
double pi = 3.14;  // 실수형 변수 선언 및 초기화
char letter = 'A';  // 문자형 변수 선언 및 초기화
boolean isTrue = true;  // 논리형 변수 선언 및 초기화
```

## 3. 형변환 (Casting)

자바에서는 형변환을 통해 변수의 자료형을 변환할 수 있다. 형변환은 **묵시적 형변환**과 **명시적 형변환**으로 나뉜다.

### 3.1 묵시적 형변환

작은 크기의 자료형에서 큰 크기의 자료형으로 변환될 때 자동으로 형변환이 이루어진다. 별도의 형변환 작업 없이 자연스럽게 데이터가 변환된다.

```
int a = 10;
double b = a;  // 묵시적 형변환: int -> double
```

### 3.2 명시적 형변환

큰 크기의 자료형을 작은 크기의 자료형으로 변환할 때 명시적으로 형변환을 해야 한다. 이때 자료의 손실이 발생할 수 있다.

```
double x = 9.99;
int y = (int) x;  // 명시적 형변환: double -> int, 소수점 아래는 제외 됨
```

## 4. 연산자

자바의 연산자는 산술 연산, 관계 연산, 논리 연산 등으로 나뉜다.

### 4.1 산술 연산자

- **덧셈**: `+`
- **뺄셈**: `-`
- **곱셈**: `*`
- **나눗셈**: `/`
- **나머지**: `%`

```
int sum = 10 + 20;  // 덧셈
int remainder = 10 % 3;  // 나머지
```

### 4.2 관계 연산자 (비교 연산자)

- `>=`, `>`, `<=`, `<`, `==`, `!=`

관계 연산자는 두 값을 비교하여 `boolean` 값인 `true` 또는 `false`를 반환한다.

```
boolean result = (10 > 5);  // true 반환
```

### 4.3 논리 연산자

- **논리곱**: `&&` (모두 참일 때 참)
- **논리합**: `||` (하나라도 참일 때 참)
- **부정**: `!` (참 -> 거짓, 거짓 -> 참)

```
boolean andResult = (10 > 5) && (5 > 1);  // true
boolean orResult = (10 > 5) || (5 < 1);  // true
boolean notResult = !(10 > 5);  // false
```

### 4.4 삼항 연산자

형식: `(조건) ? 값1 : 값2;`

조건이 참이면 값1, 거짓이면 값2가 반환된다.

```
int result = (10 > 5) ? 10 : 5;  // 10 반환
```

## 5. 배정 연산자와 단축 배정 연산자

### 5.1 배정 연산자

배정 연산자는 `=`를 사용하여 값을 변수에 할당한다.

```
int a = 10;  // 10을 a에 저장
```

### 5.2 단축 배정 연산자

단축 배정 연산자는 배정 연산을 간편하게 수행한다.

```
int a = 10;
a += 5;  // a = a + 5와 동일
```

------

## 6. 비트 연산자와 쉬프트 연산자

### 6.1 비트 연산자

비트 연산자는 이진수로 변환하여 비트 단위로 연산을 수행한다.

- **논리곱**: `&` (둘 다 1일 때만 1)
- **논리합**: `|` (하나라도 1이면 1)
- **배타적 논리합**: `^` (서로 다를 때 1)

```
int result = 10 & 7;  // 비트 연산: 1010 & 0111 = 0010, 결과는 2
```

### 6.2 쉬프트 연산자

쉬프트 연산자는 비트를 좌우로 이동시켜 연산을 수행한다.

- **왼쪽 쉬프트**: `<<` (왼쪽으로 이동, 오른쪽은 0으로 채움)
- **오른쪽 쉬프트**: `>>` (부호 비트로 채움)
- **부호 없는 오른쪽 쉬프트**: `>>>` (0으로 채움)

```
int leftShift = 10 << 2;  // 1010 << 2 = 101000, 결과는 40
```

## 7. 예제 코드

### 7.1 변수 선언 및 초기화

```
int num = 10;
double pi = 3.14;
boolean isTrue = true;
char letter = 'A';
```

### 7.2 형변환 예제

```
int a = 10;
double b = a;  // 묵시적 형변환

double x = 9.99;
int y = (int) x;  // 명시적 형변환, 소수점 이하 절삭
```

### 7.3 연산자 사용 예제

```
int sum = 10 + 20;  // 덧셈
boolean result = (10 > 5) && (5 > 1);  // 논리 연산
int ternaryResult = (10 > 5) ? 10 : 5;  // 삼항 연산자
```
